# learning-monorepo

这是我的个人工程化学习总仓库，用于系统提升 C++、Python、深度学习、联邦持续学习、游戏开发、MOD 研究和 AI Agent 开发能力。

## 学习目标

本仓库服务于 6-8 个月系统学习路线，核心目标是：

1. 补全 C++ / Python 工程化能力。
2. 建立可持续的每日代码实践和 GitHub 提交流程。
3. 完成阶段性实战项目。
4. 建立 AI 辅助学习、开发、审查和复盘闭环。
5. 最终完成 DroneRF 联邦持续学习系统、2D 射击游戏原型、个人学习 Agent 等作品集项目。

## 每日学习流程

每个学习日控制在 2-3 小时：

1. 15 分钟：复习前 3 个学习日内容。
2. 30 分钟：阅读官方资料或开源项目代码。
3. 60-90 分钟：完成可运行代码或文档产出。
4. 20 分钟：测试、重构、提交。
5. 10 分钟：AI 审查与学习日志。

## 阶段规划

### Phase 1: 工程基础

目标：掌握 Git、CMake、vcpkg、C++ 工程结构、Python 工程结构、测试、lint、CI。

综合项目：Telemetry Lab。

### Phase 2: 系统工程与游戏开发

目标：使用 raylib / Pygame / Unity / Unreal 完成斜俯视角 2D 像素射击游戏原型。

综合项目：Top-down Shooter Prototype。

### Phase 3: 深度学习、联邦学习与个人学习 Agent

目标：掌握 PyTorch、Flower、Avalanche、LangChain，并实现个人学习 Agent。

综合项目：Personal Learning Agent。

### Phase 4: Capstone

目标：完成基于 DroneRF 的联邦持续学习射频识别系统。

综合项目：DroneRF Federated Continual Learning System。

## 提交规范

每天至少完成一次有效提交。

Commit message 使用以下格式：

- chore: 初始化、配置、目录调整
- docs: 文档、学习日志、路线整理
- feat: 新功能
- fix: 修复问题
- test: 添加或修改测试
- refactor: 重构代码
