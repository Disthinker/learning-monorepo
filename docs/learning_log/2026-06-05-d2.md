## 4. 今日学到的关键点

1. 工具链验证不能只创建源码文件，还必须记录 configure、build、run 的真实结果。
2. CMake 项目最小结构包括 `CMakeLists.txt` 和 `src/main.cpp`。
3. Windows 下可以先使用 MSVC 作为 C++ 主工具链，不需要同时安装 g++ 和 clang++。
4. Python 项目后续应该统一使用 `.venv` 隔离环境。
5. commit message 中 `docs:` 后面需要保留一个空格。
