# Phase Overview

## Phase 1：工程基础阶段

核心目标：补齐现代工程化能力，建立 GitHub、构建、测试、CI、代码规范和文档习惯。

最小成果：

- 能维护一个结构清晰的学习 monorepo
- 能使用 Git 分支、commit、push、Issue
- 能创建基础 C++ / Python 工程
- 能运行测试和静态检查
- 能让 GitHub Actions 自动检查项目

## Phase 2：系统工程与游戏开发阶段

核心目标：通过游戏开发学习系统设计、资源管理、状态机、碰撞、寻路和跨语言工程对照。

## Phase 3：深度学习、联邦学习与个人学习 Agent 阶段

核心目标：掌握 PyTorch、联邦学习、持续学习和 AI Agent 的工程化实现方式。

## Phase 4：Capstone 阶段

核心目标：完成 DroneRF 联邦持续学习射频识别系统，形成可实验、可复现、可报告的综合项目。
